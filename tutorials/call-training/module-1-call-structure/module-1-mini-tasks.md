# Модуль 1. Мини‑задания для новичка

1. Прочитать вслух весь скрипт 3 раза, записать себя на аудио.

2. Отдельно потренировать только:
   - шаг 5 (перефраз цели);
   - шаг 9 (фраза «вы будете онлайн…»).

3. В паре: один играет родителя с разными запросами, второй — оператор строго по структуре 1–11 шагов.
